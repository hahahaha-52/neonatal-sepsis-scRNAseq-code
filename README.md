# Code for neonatal sepsis single-cell RNA-seq analysis

This repository contains custom scripts used for the manuscript:

**A single-cell transcriptomic atlas reveals the immune panorama of neonatal sepsis**

The scripts include preprocessing, quality control, clustering, annotation, differential expression analysis, gene set scoring, trajectory analysis, ligand-receptor analysis, pathway enrichment, and figure generation.

## Data

The scRNA-seq data are deposited in a public repository as described in the manuscript. Patient-level clinical information is not included in this repository.
